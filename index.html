기술 상세 설명
라이브러리 정리
  • pandas: 표 형태(테이블) 데이터 전처리
  • geopandas: pandas + 공간 정보(점/선/면) 지원 (Shapely 기반)
  • folium: Leaflet.js 기반의 인터랙티브 웹 지도 시각화 (HTML로 저장 가능)
  • pyproj: 좌표계 변환(투영/경위도 ↔ 평면, EPSG 변환)
  • fiona: 벡터 파일(Shapefile, GeoJSON 등) 읽기/쓰기 백엔드

  • sjoin 연산

    개념

    • 두 공간 레이어(점/선/면) 를 공간 관계(포함·교차 등)로 조인
    • SQL의 join과 비슷하지만, 키 대신 공간관계(predicate) 사용

    코드

    gpd.sjoin(left_gdf, right_gdf, how="left", predicate="intersects")
    • how: "left"(기본), "inner", "right"
    • predicate(자주 쓰는 것)
      • "intersects": 겹치기만 하면 매칭(가장 범용)
      • "within": 왼쪽 지오메트리가 오른쪽 안에 있어야 함
      • "contains": 왼쪽이 오른쪽을 포함해야 함
      • "touches": 경계만 접촉
      • "overlaps": 일부 겹치되 어느 한쪽이 완전 포함 아님
      • "crosses", "covers", "covered_by" 등도 사용 가능

        (환경/버전에 따라 지원 predicate가 다를 수 있으나 Shapely 2.x 기준 위 항목들 지원)
